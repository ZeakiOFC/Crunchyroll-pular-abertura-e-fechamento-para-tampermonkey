# Crunchyroll Pular Abertura e Fechamento Para Tampermonkey

## Pra que serve
Este script é uma extensão para o Tampermonkey que permite pular automaticamente as introduções e créditos em episódios no site da Crunchyroll, melhorando a experiência de streaming contínuo.

## Como usar
1. Instale a extensão Tampermonkey no seu navegador (disponível para Chrome, Firefox, Edge e outros). Faça o [download](https://www.tampermonkey.net/) e siga as instruções de instalação.

2. Após instalar o Tampermonkey, instale o script clicando [aqui](https://github.com/ZeakiOFC/Crunchyroll-pular-abertura-e-fechamento-para-tampermonkey/raw/refs/heads/main/crunchyroll-auto-skip.user.js). O Tampermonkey irá detectar o script e oferecer a opção de instalação.

3. Acesse o site da Crunchyroll e assista aos episódios – o script fará o skip automático das intros e créditos.
